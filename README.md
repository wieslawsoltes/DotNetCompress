An .NET compression tool.

# Usage

```
dotnet tool install --global DotNetCompress --version 1.0.0-preview.1
```

```
Usage: DotNetCompress <path> <pattern>
```

```
dotnetcompress /publish/files/path *.dll
```

```
dotnetcompress /publish/files/path *.wasm
```
